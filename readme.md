# sigfried's dotfiles. 

### The configurations I use in my workstations, both MacOS and Linux.
![](https://i.imgur.com/0431KrD.png)

## How to install:
```bash
~ > git clone https://github.com/sigfriedCub1990/dotfiles/
~ > cd dotfiles
~/dotfiles > ./install
```
